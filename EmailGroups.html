INFORMATION GUIDE

FOS

I would like to take a moment and introduce you to a new tool that we started using and have found very helpful to both sales and administration.

A new email group has been created to help facilitate any changes or request regarding any sales order. Some of you may already be using this email and that is great.

We hope to have everyone companywide benefiting from this new tool. The group name is FOS, therefore the email is fos@acetools.com

FOS was created to help both sales and verification process any changes to sales order in a timely manner. 

All you do is email the changes you need made to FOS and they will get it done and reply with a confirmation email.

 

***Subject: (Should always be the SO #) ***

Example #1 

Subject: 1234567

Body: Please remove freight & STC

Example #3

Subject: 1234567

Body: Please remove item #’s __ and add item #’s__

Example #4

Subject: 1234567 & 1234568

Body: Please change terms to __ (provide new terms)

Example #5

Subject: 1234567

Body: Please add vendors (Provide vendor name and number)


Example #6

Subject: 1234567

Body: Please change price for item #__ to $289.90

Example #7

Subject: 1234567

Body: Please change item #vip29 Qty to 2 

 

Very Important!! If more than 3 item changes or if the only one item on the order is being changed please resubmit a new order and then email FOS with the new SO# and we will backdate as needed.
